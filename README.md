# Portfolio Andry Morgan

Un portfolio moderne et responsive pour présenter mes compétences en tant que Data Analyste et développeur.

## 🚀 Aperçu

Ce portfolio présente mon parcours académique à HEI, mes compétences techniques et mes projets. Il est conçu avec une approche ergonomique et moderne, mettant l'accent sur l'expérience utilisateur.

## ✨ Fonctionnalités

- **Design Responsive** : Optimisé pour tous les appareils (mobile, tablette, desktop)
- **Animations au Scroll** : Effets d'animation fluides lors du défilement
- **Navigation Mobile** : Menu hamburger pour les petits écrans
- **Formulaire de Contact** : Redirection vers Gmail avec contenu pré-rempli
- **Accessibilité** : Labels ARIA et contraste optimisé
- **Performance** : Chargement rapide et optimisé

## 🛠️ Technologies Utilisées

### Frontend
- **HTML5** : Structure sémantique
- **CSS3** : Design moderne avec variables CSS
- **JavaScript** : Interactivité et animations

### Fonctionnalités
- **Intersection Observer** : Détection du scroll pour les animations
- **Responsive Design** : Media queries et flexbox/grid
- **Form Validation** : Validation côté client
- **Smooth Scrolling** : Navigation fluide

## 🔧 Personnalisation

### Modifier les couleurs
Dans `assets/css/style.css`, modifiez les variables CSS :
```css
:root {
    --primary: #votre-couleur;
    --secondary: #votre-couleur;
    /* ... */
}
```

### Ajouter un projet
Dans `index.html`, ajoutez une nouvelle carte dans la section projets :
```html
<article class="project-card">
    <div class="project-header">
        <h3>Votre Projet</h3>
        <span class="project-badge">Type</span>
    </div>
    <p class="project-description">Description...</p>
    <div class="project-tech">
        <span class="tech-tag">Tech1</span>
        <span class="tech-tag">Tech2</span>
    </div>
</article>
```

### Modifier les compétences
Ajoutez de nouvelles catégories ou compétences dans la section skills.
